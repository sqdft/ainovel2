# 小羊老师的小说生成器 (Teacher Xiaoyang's Novel Generator)

这是一个基于人工智能的强大网文小说生成工具，专为创作者设计。无论是百万字的长篇巨制，还是几万字的爆款短篇，都能通过本工具轻松生成高质量的大纲、角色设定和正文内容。

## 🌟 核心功能

### 📖 长篇小说生成模式
*   **智能大纲与世界观**：输入书名和核心脑洞，一键生成完整的故事大纲和世界观设定。
*   **角色设定系统**：自动生成包含主角、配角、反派在内的详细人物小传（性格、背景、核心冲突）。
*   **章节目录批量生成**：支持按批次生成章节目录，并带有**智能完结倒计时机制**：
    *   **开篇/发展期**：稳步推进剧情，铺陈世界观。
    *   **高潮期**：围绕大纲展开核心冲突。
    *   **收网期**：距离完结10章内，自动触发收网警告，解决次要矛盾。
    *   **大结局**：最后一章触发红色完结警告，完美填坑，拒绝烂尾。
*   **正文连载生成**：根据章节摘要和前文剧情，生成符合网文节奏的高质量正文（支持断章悬念设置）。

### ⚡ 爆款短篇生成模式
*   **网感标题生成**：选择主题（如：真假千金、追妻火葬场、规则怪谈等），AI 化身“爆款网文主编”，为你生成极具吸引力、点击率极高的爆款标题。
*   **短篇大纲构思**：根据选定的标题和主题，生成 7k-12k 字短篇小说的核心脑洞和详细大纲。
*   **正文一键生成**：一气呵成生成短篇正文，情绪拉扯到位，反转拉满。

### 🛠️ 通用便捷功能
*   **多主题选择**：内置海量网文热门主题，支持多选组合。
*   **多模型支持**：支持接入 Gemini、DeepSeek、OpenAI、Anthropic 等多种主流 AI 大模型。
*   **一键导出**：生成的小说支持一键下载为 `.txt` 文本文件，方便后续编辑和发布。

## 🚀 快速开始

1.  **配置 API Key**：在“设置”面板中选择你喜欢的 AI 模型服务商，并填入对应的 API Key。
2.  **选择创作模式**：在顶部导航栏选择“长篇小说”或“短篇故事”。
3.  **填写基础设定**：选择主题、输入脑洞/简介。
4.  **开始生成**：按照页面引导，依次生成大纲 -> 角色 -> 目录 -> 正文。
